Installation
------------

1. Add middleman wundertuete to your Gemfile:

    ```
    group :middleman do
      gem 'middleman-wundertuete', github: 'bitaculous/middleman-wundertuete'
    end
    ```

2. Run `bundle install`

3. Activate the “wundertuete” in your middleman config:

    ```
    activate :wundertuete
    ```
